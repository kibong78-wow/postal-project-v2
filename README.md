# 📮 우정사업본부 업무 지식베이스

우체국 현업 지도실장을 위한 법령·복무·징계 실무 지원 시스템입니다.

---

## 주요 기능

| 탭 | 기능 |
|---|---|
| 홈 | 주요 통계, 빠른 검색, 중요 카드 요약, 법령 목록 |
| 업무카드 | 30개 실무 카드 검색·필터 (복무·근무·휴가·괴롭힘·감사·징계) |
| 법령·조문 | 7개 법령 52개 조문 내장 DB, 조문 단위 검색 |
| AI 분석 | 상황 입력 → Claude AI가 법령·사례 기반 종합 분석 |
| 사례등록 | 이미지 업로드 → AI가 민감정보 자동 마스킹 후 사례 DB 저장 |
| 체크리스트 | 지점별·일자별 점검 세션 구성·실행·기록·조회 |
| DB 업데이트 | 변경 이력 관리 및 업데이트 요청 문구 자동 생성 |

## 로컬 개발 실행

### 사전 요구사항
- Node.js 18 이상

### 설치 및 실행

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행 (http://localhost:3000)
npm run dev

# 3. 프로덕션 빌드
npm run build

# 4. 빌드 결과 미리보기
npm run preview
```

## Vercel 배포 (권장)

### 방법 1 — GitHub 연동 자동 배포

```
1. GitHub에 이 폴더 전체를 새 저장소로 업로드
2. vercel.com → New Project → 해당 저장소 선택
3. Framework Preset: Vite 선택 (자동 감지됨)
4. Deploy 클릭 → 1~2분 후 URL 생성
5. 이후 GitHub에 파일 교체 → Vercel 자동 재배포
```

### 방법 2 — Vercel CLI

```bash
npm install -g vercel
vercel login
vercel --prod
```

## DB 업데이트 방법

1. 앱의 **DB 업데이트** 탭 접속
2. 변경할 항목 입력 (추가/수정/삭제)
3. **"복사 후 이력 등록"** 클릭 → 클립보드에 요청문 복사
4. Claude 대화창에 붙여넣기 → 새 `App.jsx` 파일 수령
5. `src/App.jsx` 파일 교체 → GitHub에 업로드 → Vercel 자동 재배포
